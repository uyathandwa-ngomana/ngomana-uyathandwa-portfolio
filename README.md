# My Developer Portfolio 🚀

This is my personal portfolio website, built to showcase my projects and skills as I finish my final year in ICT Application Development at CPUT.

### 🛠️ Built With:
- **HTML5** (Structure)
- **CSS3** (Styling & Animations)
- **Vanilla JavaScript** (Interactivity)
- **Boxicons** (Icons)
- **Google Fonts** (Typography)

### ✨ Features:
- **Dark/Light Mode**: Fully functional toggle for different viewing preferences.
- **Responsive Design**: Works on mobile, tablets, and desktops.
- **Scroll Animations**: Smooth reveal effects as you browse the site.
- **Live CV**: Direct link to my latest professional CV.

### 📂 How to Run Locally:
1. Download or clone this repository.
2. Make sure `index.html`, `style.css`, `script.js`, `profile.webp`, and `UYATHANDWA_CV.pdf` are all in the same folder.
3. Open `index.html` in any web browser.
4. (Optional) Use the **Live Server** extension in VS Code for real-time updates while editing.

### 🚀 How to Deploy:
I used **GitHub Pages** to host this site. To do the same:
1. Create a new public repository on GitHub.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Set the branch to **main** and click **Save**.
5. Your site will be live at `https://yourusername.github.io/repository-name/`.
