Featured Projects

DailyWin — Task Management App

A task management app built with Laravel 12, supporting two user roles: admin and team member. Users can create, assign, and track tasks through a status workflow from pending to in progress to completed, organize them by category and priority, and get automatic deadline reminders. Admins get a dashboard with full task and user oversight.

My role was the backend logic: authentication with Laravel Breeze, role-based access control, and the controllers handling task, category, and user management.

Team: Amanda Msutu, Zahrah Vermaak, Uyathandwa Ngomana
